# SkillCraftRPG API

SkillCraft is a Tabletop Role-Playing Game.

## Environment Variables

All the following environment variables are optional.

- `CACHING_ACTOR_LIFETIME`: the lifetime of cached actors. Defaults to `00:15:00`.
- `PERMISSION_WORLD_LIMIT`: the maximum number of worlds a user may create. Defaults to 3.
- `POSTGRESQLCONNSTR_SkillCraft`: the PostgreSQL connection string.
